# ĀYĀT · آيات — The Quran as a Universe of Signs

One self-contained HTML file. Every one of the Quran's **6,236 verses is a star**;
the **114 surahs** are phyllotaxis clusters along a two-armed spiral galaxy
(warm gold = Meccan, cool blue = Medinan; the long early surahs form the
luminous galactic core). Selecting any of the **199 topics** ignites its verses
across the whole galaxy and threads them, in mushaf order, into a constellation.

## Use
- **Open `index.html`** in any modern browser (Chrome/Edge/Firefox/Safari,
  desktop or Android/iOS). Works fully offline — everything is embedded.
- **Host it publicly**: drop the file on GitHub Pages, Netlify Drop, Cloudflare
  Pages, or any static host. No build step, no server code.

## Controls
Drag to orbit · wheel / pinch to zoom · click a star to read its verse ·
`/` focuses search (try `2:255`, `Yusuf`, `tawakkul`, or English full-text) ·
✦ Topics opens the 14-branch constellation index · ⟡ Tour flies star-to-star
through a topic · EN / اردو / हिं switches the translation · Esc clears.

## Topic index — method
Topics were generated by **root-form concordance against the actual corpus**
(diacritic-stripped Uthmani; e.g. Ṣabr matches every ص-ب-ر form) plus curated
verse ranges for narratives, then validated against canonical concordance
counts (Mūsā 131 name-verses, Ibrāhīm 63, ṣabr 96 …). Roots can serve more
than one sense, so treat it as a study companion. The full index is
**editable**: `ayat_topics.json` ships alongside, and the About panel inside
the app offers the same download.
